# 模块4：进阶知识：机器学习与AI大赛

本模块深入传统机器学习算法，培养AI竞赛能力和分析式AI项目开发技能。

## 课程内容

### 课程18：分析式AI基础
- 分析式AI与生成式AI的区别
- 十大经典机器学习算法详解
- 贝叶斯分类器、决策树、随机森林、SVM、逻辑回归
- **项目实战**：二手车价格预测，完整数据建模流程

### 课程19：不同领域的AI算法
- 金融行业AI应用（客户流失、风控模型、期货套利）
- 制造业缺陷检测
- 快消行业供应链优化
- **AI大赛进阶**：二手车价格预测竞赛优化

### 课程20：时间序列AI大赛
- 时间序列分析基础（AR、MA、ARMA、ARIMA）
- statsmodels工具应用
- **实战项目**：沪市指数预测、资金流入流出预测、周期因子分析

## 学习目标

通过本模块学习，您将：
- 掌握传统机器学习算法的核心原理和应用场景
- 具备跨行业AI解决方案设计能力
- 培养AI竞赛建模思维和实战技巧
- 精通时间序列分析与金融数据建模
